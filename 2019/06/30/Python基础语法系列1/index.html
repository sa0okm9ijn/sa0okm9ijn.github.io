知识四:数据类型

最常用的数据类型有三种——字符串(str)、整数(int)和浮点数(float)

字符串

只要是被【单/双/三引号】这层皮括起来的内容，不论那个内容是中文、英文、数字甚至火星文。只要是被括起来的，就表示是字符串类型。

整数

是正整数、负整数和零的统称，是没有小数点的数字

浮点数

带小数点的数字，就是浮点数

知识五:type()查看数据类型
